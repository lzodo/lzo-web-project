CSS选择器
伪类与伪元素
  • 1、规范中伪类一个冒号表示，伪元素两个冒号，为了兼容有些伪元素也能用单冒号
  • 2、常见伪类选择器如 ：hover、:first-child、：nth-child(2n)等
  • 3、常见伪元素选择器如 E::selection、E::after、E::first-letter等
属性选择器
  • 1、与正则类似、用中括号
  • 2、E[attr*="val"] 选择具有att属性且属性值为包含val的字符串的E元素
关系选择符
  • 1、包含E F | 子 E>F
  • 2、E紧贴的下一个F E+F | E后面所有同级F元素 E~F
元素选择符
  • 1、 * div #xxx .xxx 等
属性
颜色表示方法
  • p{color:green;}
    p{color:#ff0000;}
    p{color:rgb(0,0,0);}
    下面的IE8一下不支持
    p{color:rgba(0,0,0,.5);}
    p{color:hsl(240,50%,50%);} 色调、饱和度、亮度
    p{color:hsla(240,50%,50%,.5);}
    p{color:transparent;} 透明色
text-shadow文本阴影
  • 1、text-shadow:none| px{2,3} && color,px{2,3} && color2
  • 2、参数:左右偏移,上下偏移,模糊程度,颜色
text-overflow文本溢出省略号
  • /*单行*/
    overflow: hidden;
    text-overflow: ellipsis;
    white-space:nowrap;
  • /*多行溢出省略号显示 不支持火狐*/
    overflow: hidden;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 1;
    -webkit-box-orient: vertical;
border-radius圆角边框
  • 四参数:上左 - 上右 -下右 - 下左
  • 三参数:上左 - 上右,下左 - 下右
  • 双参数:上左,下右 - 上右,下左
  • 一参数:四个角
  • border-radius:5px/10px 水平5像素，垂直10像素
  • border-top-right-radius:5px/10px 上右角水平5像素，垂直10像素
box-shadow边框阴影
  • 1、box-shadow:none|inset? [ px{2,4} && color ], [ px{2,4} && color ]
  • 2、px参数:水平偏移,垂直偏移,模糊程度,向外延申
border-image
  • 1、主要三个参数:路径(source)、切割(slice)、平铺(repeat)
  • 2、切割的fill决定盒子背景要不要用图片除边框宽度的中间部分填充
  • 3、border-image-repeat : [ stretch | repeat | round ]{1水平,2垂直}
  • 4、border-image: url(/border-image.png) 27 fill repeat repeat;
outline
  • 1、与边框一样 outline:1px solid #000;
  • 2、css3新增 outline-offset: px 设置偏移可负数
  • outline-offset:-10px
transition 过渡
  • transition：all .5s ease 1s

  • 1、transition-property:all; 过渡的属性
  • 2、transition-duration:.5s; 过渡时间
  • 3、transition-timing-function:ease; 运动类型(贝塞尔曲线)
  • 4、transition-delay:1s; 延迟时间
animation 动画
  • animation：name .5s ease 1s;

  • 1、animation-name:name; 动画名称
  • 2、animation-duration:.5s; 动画持续时间
  • 3、animation-timing-function:ease; 运动类型(贝塞尔曲线)
  • 4、animation-delay:1s; 延迟时间
  • 5、animation-iteration-count:5; 循环次数(infinite无限循环)
  • 6、animation-direction :normal; 是否反向运动(alternate与反向交替)
背景
background背景
  • background:颜色 url 平铺 滚动 定位
  • 1、滚动:background-attachment:fixed(跟窗口)||local(跟元素内容)||scroll(跟元素)
background-origin背景访问
  • background-origin:border-box; 背景从边框开始
  • background-origin:content-box; 背景从内容开始
  • background-origin:padding-box; 背景从padding开始(默认)
background-clip裁剪
  • background-clip:border-box; 从border向外裁剪(不含border)
  • background-clip:content-box; 从content向外裁剪(不含content)
  • background-clip:padding-box; 从padding向外裁剪(不含padding)
background-size背景大小
  • background-size:number; 像素设置大小
  • background-size:50%; 百分比设置大小
  • background-size:auto; 原图大小
  • background-size:cover; 将背景图像等比缩放到完全覆盖容器，背景图像有可能超出容器
  • background-size:contain; 将背景图像等比缩放到宽度或高度与容器的宽度或高度相等，背景图像始终被包含在容器内。
  • 合并到background:background-position 和 background-size 之间用 / 分开。
多重背景
  • background:url(test1.jpg) no-repeat scroll 10px 20px,url(test2.jpg) no-repeat scroll 50px 60px
  • 用逗号隔开,颜色无法设置多重
  • 拆分方式也是用逗号隔开
语法规则
@import
  • @import url("global.css");
  • @import url(global.css);
  • @import "global.css"
  • 以上3种方式都有效。当使用url(url)的方式时，包住路径的引号可有可无；当直接使用url时，包住路径的引号必须保留
  • 媒体查询导入实例
  • @import url(example.css) screen and (min-width:800px);
@charset
  • 在外部样式表文件内使用。指定该样式表使用的字符编码。
  • 格式:@charset "utf-8";
  • 后面分号不能省略
@media
  • @media:设备类型 and (媒体特性) and (媒体特性){}
  • @media和@import规则上，也可以被用在HTML和XML中 
    
@media screen and (width:800px){ … }
@import url(example.css) screen and (width:800px);
<\link media="screen and (min-width:800px) and (max-width:900px)" rel="stylesheet" href="example.css" />
<\?xml-stylesheet media="screen and (width:800px)" rel="stylesheet" href="example.css" ?>
@font-face
  • 定义字体 @font-face{font-family:字体名称;src:字体路径;sRules;}
  • 定义 
    @font-face{
  font-family:YH;
  src:url(http://domain/fonts/MSYH.TTF);
}
  • 使用 
    body{font-family:'YH';}
@page
@keyframes
  • 定义 : @keyframes 动画名称{动画每个节点的动作}
  • 使用 : el{animation: 动画名称 3s linear;}
  • @keyframes定义的动画名称用来被animation-name所使用
  • 必须添加前缀Safari 和 Chrome(-webkit-)、Firefox(-moz-)、Opera(-o-)
  • 动作写法1 
    @keyframes 动画名称{
  from{opacity:1;}
  to{opacity:0;}
}
  • 动作写法2 
    @keyframes testanimations{
  from或0%{transform:translate(0,0);}
  20%{transform:translate(20,20);}
  40%{transform:translate(40,0);}
  60%{transform:translate(60,20);}
  80%{transform:translate(80,0);}
  to或100%{transform:translate(100,20);}
}
